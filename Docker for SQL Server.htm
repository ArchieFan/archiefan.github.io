Download SQL Server Linux images

 

docker pull mcr.microsoft.com/mssql/server:2019-latest

 

docker pull mcr.microsoft.com/mssql/server:2017-latest

 

 

Run the docker images

 

docker run --name sql_2017 -e "ACCEPT_EULA=Y" -e "SA_PASSWORD=1Secure*Password1" -e "MSSQL_PID=Enterprise" -e “ MSSQL_AGENT_ENABLED=True ” -p 1433:1433 -d mcr.microsoft.com/mssql/server:2017-latest

 

docker run --name sql_2019_1436 -e "ACCEPT_EULA=Y" -e "SA_PASSWORD=1Secure*Password1" -e "MSSQL_PID=Enterprise" -e “ MSSQL_AGENT_ENABLED=True ” -p 1436:1433 -d mcr.microsoft.com/mssql/server:2019-latest

 

 

 

To test connection

docker exec -it sql_2017 /opt/mssql-tools/bin/sqlcmd -S localhost -U sa

 

docker exec -it sql_2019_1436 /opt/mssql-tools/bin/sqlcmd -S localhost -U sa

 

 

Connect to SQL Server instance using SSMS

 

Connect to the SQL Server 2017 instance using the default port, 1433, on the localhost.
